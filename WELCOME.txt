┌─────────────────────────────────────────────────────────────────────────────┐
│                       🐛 TROUBLESHOOTING                                     │
└─────────────────────────────────────────────────────────────────────────────┘

  Problem: Can't start server
  Solution: Install Python 3.x from python.org
  
  Problem: Database not found
  Solution: Run "bash scripts/init_db.sh"
  
  Problem: Permission denied
  Solution: Run "chmod +x run.sh scripts/*.sh"
  
  Problem: Port already in use
  Solution: Change PORT in run.sh to another port (e.g., 3000)
  
  Full troubleshooting guide available in INSTALLATION_GUIDE.md
